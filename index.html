Theme Decks

Theme decks are available up to Sword and Shield: Vivid Voltage.

To access the theme decks, use the endpoint 

/theme-decks/<setid>

Example: Sun & Moon: Guardians Rising Theme Decks

Images

Currently, only set logo and symbols are hosted with this API and card images are linked to This API. Expect to see image support hosted for all cards in a few weeks.
